# vLLM-Structure
This project is developed for Structured Radiology Reporting at level-2 based on [vLLM](https://github.com/vllm-project/vllm).

## Installation

```angular2html
git clone https://github.com/niuchuangnn/vllm_structure.git
cd vllm_structure
pip install -e .
```
